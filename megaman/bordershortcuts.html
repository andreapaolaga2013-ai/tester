Border Shortcuts
Simply styling border: black; will default to a solid black border with a 3px width/thickness.
If you wanted to, you could also style for each side of a border. See the style section.
You can also pick and choose which side you want to style. See the style section.
Maybe you want just the left and right side instead of top and bottom.
You can also just use number positioning. This box was styled with solid black 3px. Using only one number will make all four sides the same. See the style section.
Styling with four numbers will individually control the four border. Each number corresponds to a certain spot. The first number goes with the top border, the second with the right border, the third with the bottom, and the fourth with the left. It goes clockwise starting with the top. See the style section. All you have to do in the style tag is declare border-width.
If you use border-width, but only with the three numbers, the first number will control the top, the second number will control BOTH the LEFT and the RIGHT, and the third number will control the bottom. See the style tag.
If you use border-width, but only with two numbers, the first number will control the top and bottom, and the second number will control both the left and right. See the style tag.
